HTML

Hypertext Markup Language'ın kısaltmasıdır. HTML, web sayfalarının yapısını ve içeriğini tanımlamak için kullanılan bir işaretleme dilidir. Web tarayıcıları tarafından anlaşılabilen bir formatta yazılan HTML belgeleri, web sayfalarının görüntülenmesini sağlar.

HTML, metin, resimler, bağlantılar, tablolar, form alanları ve diğer çeşitli öğeleri içerebilen yapısal bir dildir. Belge içinde etiketler kullanarak içeriği işaretler ve yapılandırır. Her bir etiket genellikle açılış () ve kapanış () şeklinde iki bileşenden oluşur. Etiketler, içeriklerin nasıl görüntüleneceğini belirtir veya belgeye özel işlevler ekler.

CSS

CSS, Cascading Style Sheets'in kısaltmasıdır. CSS, web sayfalarının görünümünü ve düzenini kontrol etmek için kullanılan bir stil dilidir. HTML veya diğer işaretleme dilleriyle birlikte kullanılarak, web sayfalarının renkleri, fontları, boyutları, yerleşimi ve diğer görsel özellikleri belirlenir.

JavaScript

JavaScript, web tarayıcılarında çalışabilen bir programlama dilidir. İnteraktif web sayfaları oluşturmak ve web uygulamalarını geliştirmek için kullanılır. JavaScript, HTML ve CSS ile birlikte kullanılarak web sayfalarına dinamik özellikler eklemeyi sağlar.

JavaScript, tarayıcı tarafında çalıştığı için istemci tarafı (client-side) bir programlama dilidir. Web sayfasının HTML yapısına erişebilir ve sayfadaki öğelerle (butonlar, formlar, metin alanları vb.) etkileşim kurabilir. Ayrıca, web tarayıcısının sağladığı API'ler ve özellikler aracılığıyla kullanıcının eylemlerine tepki verebilir, verileri işleyebilir ve sunucu ile iletişim kurabilir.

JavaScript, birçok farklı amaç için kullanılabilir. Sayfa içindeki öğelerin görünürlüğünü değiştirebilir, animasyonlar oluşturabilir, kullanıcı girişlerini doğrulayabilir, veri işleme ve manipülasyonu yapabilir, sunucuyla veri alışverişi yapabilir ve dinamik içerikleri güncelleyebilir. Ayrıca, JavaScript'in popüler web framework'leri ve kütüphaneleri (örneğin React, Angular, Vue.js) sayesinde daha karmaşık web uygulamaları da geliştirilebilir.

React JS

React, bileşen tabanlı bir yapıya sahiptir. Web sayfasını birden fazla bağımsız bileşene ayırarak her bir bileşeni ayrı ayrı yönetmeyi sağlar. Her bir bileşen, kendi içinde durum (state) ve davranış (behavior) tutabilir ve bileşenler arasında veri iletişimi sağlamak kolaydır. Bu yapı, kodun yeniden kullanılabilirliğini artırır ve büyük projelerin yönetimini kolaylaştırır.

NodeJS

Node.js, JavaScript tabanlı bir çalışma zamanı ortamıdır. Sunucu tarafında uygulama geliştirmek için kullanılır. Node.js, web sunucusu gibi sunucu tarafı işlevlerini gerçekleştirmek için JavaScript'i kullanmanıza olanak tanır.

Node.js, öncelikle tarayıcı dışında JavaScript'in çalışabilmesini sağlar. Node.js, V8 JavaScript motorunu temel alır ve JavaScript kodunu sunucu tarafında çalıştırır. Bu da JavaScript'in sadece tarayıcıda değil, sunucu tarafında da kullanılabilmesini sağlar.

Node.js, paket yöneticisi olan npm (Node Package Manager) ile birlikte gelir. npm, geliştiricilerin hazır paketleri kullanarak projelerini hızlı bir şekilde genişletmelerine ve yönetmelerine olanak tanır. npm, milyonlarca açık kaynaklı paketi içeren geniş bir ekosisteme sahiptir.
